# Shiki Svelte

A component that renders code with [Shiki](https://github.com/shikijs/shiki) in Svelte.

## Installation

```shell
npm install @aarondeloach/shiki-svelte
```

## Usage

```javascript
<script>
    import CodeHighlighter from '@aarondeloach/shiki-svelte'
</script>

<CodeHighlighter code={`<h1>Hello World</h1>`}/>
```
